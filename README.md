## Environment Setup
```bash
git clone https://github.com/brianteh/muteable-nft.git
npm i --production
```

## To run the app
```bash
cd current_folder_files
npm run dev
```
## Before pushing to origin
```bash
npm run build 
npm run start
```
If the build failed, please do not push to the main branch.
